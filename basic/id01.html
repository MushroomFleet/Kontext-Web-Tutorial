FLUX Kontext Prompt

Instruction-Based Editing Prompt

Why This Prompt Works

This prompt is effective because it uses a clear action verb ("Change") followed by a specific target ("the car") and a detailed description of the desired outcome ("vibrant metallic green"). The specificity of "metallic green" helps FLUX Kontext understand both the color and finish properties to apply.
