FLUX Kontext Prompt

Object Removal
Remove the person in the background

Why This Prompt Works

This prompt is effective because it clearly specifies the action ("Remove") and precisely identifies the target ("the person in the background"). The location specification "in the background" helps FLUX Kontext distinguish between multiple people who might be present, ensuring only the unwanted background figure is removed while preserving main subjects and foreground elements.
